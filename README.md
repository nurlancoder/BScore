# BScore - Business Stability Score App

Bu proje, işletme istikrar skorunu hesaplayan ve görselleştiren bir React Native mobil uygulamasıdır.

## Özellikler

- **Ana Sayfa**: Ana ekran ve navigasyon
- **BS Score 1**: İşletme ID giriş ekranı
- **BS Score 2**: İşletme tipi seçim ekranı
- **BS Score 3**: İşletme aktivite detayları ekranı
- **BS Score 4**: QR kod tarama ekranı
- **BS Score 5**: Hesaplama/İşleme ekranı
- **BS Score 6**: İşletme istikrar skoru sonuç ekranı (Progress arc ve metrikler)

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
npm start
```

## Teknolojiler

- React Native
- Expo
- React Navigation
- React Native SVG
- Poppins Font Family

## Notlar

- Tüm fontlar Poppins ailesinden kullanılmaktadır
- SVG ikonlar assets klasöründe bulunmaktadır
- Mock data ile çalışmaktadır
